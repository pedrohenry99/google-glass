What is it

O Google Glass é um acessório em forma de óculos que possibilita a interação dos usuários com diversos conteúdos em realidade aumentada. Também chamado de Project Glass, o eletrônico é capaz de tirar fotos a partir de comandos de voz, enviar mensagens instantâneas e realizar vídeo­confe­rên­cias. Seu lançamento está previsto para 2014, e seu preço deve ser de US$ 1,5 mil. Atualmente o Google Glass encontra-se em fase de testes e já possui um vídeo totalmente gravado com o dispositivo. Além disso, a companhia de buscas registrou novas patentes anti-furto e de desbloqueio de tela para o acessório.

    
Git/GitHub

1- Configurar user GitHub no VSCode:
1.1- No VSCode, terminal, new terminal, seta pra baixo "select default profile", seleciona GitBash, com o terminal aberto configurar email e user:
    git config --global user.email "email@gmail.com"
    git config --global user.name "usuárioGitHub"
2- Abrir repositorio do PC, clicar com o direito e ir em "GitBash Here". 
2- Dar os comandos abaixo:
    git init -> inicia o repositorio
    git add . -> Envia para área de staged, SALVAR arquivo no VSCode antes de dar esse comando!
    git status -> verifica como está o Git
    git commit -m "first commit" -> Commita as novas atualizações do Git no repositório da maquina
    git branch -M "main" -> Renomeia branch master para main
    git remote add origin https://github.com/pedrohenry99/google-glass.git -> Link tirado do GitHub, vai sincronizar repositório local com o GitHub
    git push -u origin "main" -> Empurra alterações do repositório local para o GitHub


    
Criar repositório no GitHub

    1- Repository -> New -> Prencher os dados

    
Criar Branch

Branch: Novas linhas de tempo
    git checkout -b "novo-botao" -> Cria uma nova branch chamada "novo botão", saindo da branch "main"
    git add . -> Envia para área de stand
    git commit -m "novo botao" -> Commita as novas atualizações
    git push origin novo-botao
    git checkout main -> volta para a branch main
    git merge novo-botao -> vai colocar a branch "novo-botao" na branch principal "main"

    
Importar Repositório GitHub no PC

    1- Crie uma pasta no PC, clica com o direito e vai em "git bash here"
    git clone "https://github.com/pedrohenry99/ProjetoGit.git" -> Clona repositório do GitHub no PC

    
Atualizar repositorio já importado pelo GitBash

    1- Abrir o Git dentro do repositório do PC (mesmo passo 1 da etapa anterior)
    git pull -> puxa as atualizações do repositório no GitHub

    
Atualizar repositorio pelo VSCode

    1- Abrir o repositório no VSCode
    2- Source Control, Stage Changes, Commit, Sync


Data de lançamento

Não há uma data específica e oficial para o dispositivo ser lançado, ainda. Pode ser que ele esteja disponível em demonstrações a partir de 2013, mas seu lançamento para as lojas fica para, pelo menos, 2014.

    - Tela:Resolução equivalente a tela de 25"

    - Conectividade: Wi-Fi/ Bluetooth

Como funciona

De acordo com fontes próximas do Google, os óculos vão contar com uma pequena tela de LCD ou AMOLED na parte superior e em frente aos olhos do usuário. Com o uso de uma câmera e GPS, você pode se situar, assim como selecionar opções com o movimento da cabeça

O que você pode fazer com o Google Glass

O vídeo de divulgação do Google mostra que você pode se transformar em uma espécie de “super-humano”, já que o aparelho pode indicar a quantos metros você está de seu destino, se o metrô está aberto ou fechado, mostrar o clima, agenda e até mesmo permitir que você marque encontros apenas com comandos de voz.

Novidades no Glass

O Google enfim revelou as especificações completas do Google Glass, e com ele uma surpresa ainda inédita no mercado: a gigante das buscas usará um sistema de áudio baseado na transdução por condução. Através das hastes dos óculos, o som será transmitido para o ouvido do usuário por meio de microvibrações em determinados ossos de sua cabeça, sem usar nenhum tipo de alto-falante.

Além da surpresa do áudio, a tela montada a frente do olho do usuário também chamou atenção. Serão 640 x 360 pixels de resolução que, em proporção, equivaleria a um monitor de 25 polegadas de alta definição colocado a 2,5 metros de distância do espectador.
